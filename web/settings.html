CAT rig connection

Polls the rig's VFO-A frequency and displays it live in the status bar. Keeps the ALL.TXT log accurate without manual dial-frequency entry.

Disabled

Windows: COM6 · Linux: /dev/ttyUSB0 · macOS: /dev/cu.usbserial

⚠ rigctld must be running before you enable CAT in this mode. Start it with e.g. rigctld -m <model-id> -r COM6 -s 9600. The daemon will retry automatically if rigctld is not yet available.

How often to query the rig. Range: 1–60 seconds.
