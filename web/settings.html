Advanced Decoder Settings

Default values are calibrated by R&R study — change with caution. Changes take effect on the next decode cycle without a daemon restart.

Minimum sync score for pass-1 candidates [5–30]. Higher = fewer false positives; lower = more sensitivity. Default: 10 (D-009 calibrated).

Normalised correlation gate for OSD candidates [0.05–0.40]. Default: 0.10 (D-009 calibrated).

Maximum Hamming distance for OSD candidates [30–100]. Default: 60 (D-009 calibrated).

Remote access

When enabled, the web interface is reachable from other devices on your local network (0.0.0.0:<port>). A daemon restart is required for binding changes to take effect.

Non-loopback clients must supply this passphrase via the X-Api-Key header (REST) or ?key= query parameter (WebSocket). Leave empty to allow unauthenticated LAN access. Stored as plaintext in app.json.

⚠ A restart is required for binding changes to take effect.
